La Gerente de Marketing María González necesita respuestas que cruzan múltiples tablas de techstyle_db. Esta semana practican JOINs: la herramienta fundamental para combinar información de tablas relacionadas.

5 Actividad Extra: EXPLAIN en la Consulta 4

EXPLAIN
SELECT pr.nombre, pr.categoria, SUM(oi.cantidad) AS total_vendido
FROM order_items oi
INNER JOIN pedidos   p  ON oi.id_pedido   = p.id_pedido
INNER JOIN productos pr ON oi.id_producto = pr.id_producto
WHERE p.estado = 'entregado'
GROUP BY pr.id_producto, pr.nombre, pr.categoria;
